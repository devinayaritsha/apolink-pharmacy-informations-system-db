# Apolink - Pharmacy Information System Database

Apolink is a personal database project built using PostgreSQL to simulate pharmacy operational processes.

## Features

- Patient transactions
- Product inventory management
- Supplier management
- Goods receiving transactions
- Stock monitoring
- Stock opname
- Reporting and analytics

## Database Tables

### Master Data

- pasien
- produk
- supplier

### Transactions

- transaksi
- detail_transaksi
- penerimaan_barang
- detail_penerimaan

### Inventory

- stok_opname

## Sample Reports

- Patient transaction history
- Product sales report
- Critical stock report
- Supplier purchase report
- Best-selling products report

## Technology

- PostgreSQL
- pgAdmin 4
